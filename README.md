# SiddusBank Backend API 
 
A comprehensive banking application backend built with Spring Boot. 
 
## Features 
- JWT-based authentication 
- User registration and login 
- Account management (create, view, delete) 
- Deposit, withdraw, and transfer money 
- Transaction history 
- Profile management 
 
## Technology Stack 
- Java 17 
- Spring Boot 3.2.0 
- Spring Security 
- JWT 
- MySQL 
- Maven
