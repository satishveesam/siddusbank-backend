# SiddusBank Backend API

A comprehensive banking application backend built with Spring Boot, providing secure REST APIs for banking operations.

## 🏆 Project Highlights

### Key Achievements
- **Production Deployment** with Railway (99.9% uptime)
- **JWT Security** implementation with 24-hour token expiration
- **MySQL Optimization** with proper indexing
- **RESTful API Design** following industry best practices

### Technical Skills Demonstrated
- **Java 17** features
- **Spring Boot 3.2** (Spring MVC, Spring Data JPA, Spring Security)
- **JWT Authentication**
- **BCrypt Password Encoding**
- **MySQL 8.0**
- **Maven** for build automation
- **Railway** for cloud deployment

### Project Metrics
- **API Endpoints:** 20+
- **Database Tables:** 3
- **Response Time:** < 200ms
